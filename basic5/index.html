table 만들기(표만들기)

html에서 표를 구현 할 때

  • tr(table row):행
  • th(table heading):행, 열의 제목(굵은체, 가운데 정렬)
  • td(table data):(행의 내용) 열
  • 속성 colspan="2":2칸의 열 병합
  • 속성 rowspan="2":2칸의 행 병합
